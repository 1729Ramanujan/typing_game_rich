# ①課題名
ポケモンタイピングゲーム（裏）

## ②課題内容（どんな作品か）
- 第１ステージはポケモン関連単語のタイピングゲームです
- 第１ステージをクリアするとパチモノのポケモン戦闘ができます

## ③工夫した点・こだわった点
- 工夫した点：
  ポケモンバトル自体は比較的簡単にできました。ボタンを押したらゲージが下がっていって、相手からもランダムで攻撃が来る様に設定するところまでは1人でも書けました。
  しかし、手持ちを複数体にしようとした時にコードをガラッと変えなければいけませんでした。
  ・あらかじめポケモンに関するデータ（HP,攻撃,防御など）をjs上に保存しておいて、そこから手持ちのポケモンの配列にそのデータを移す
  ・そのポケモンが使う技の関数を別に定義しておいて、コードの量を減らす
  ・どのポケモンが出てきたかに応じてボタンに表示される技名を変える
  ・ポケモンの画像も出てきたポケモンに応じて変える

## ④難しかった点・次回トライしたいこと（又は機能）
- 全体的なクオリティがまだまだと感じました。明日の発表までには新ポケモンと技を追加して、UIもこだわります。
